# Phân Tích Dữ Liệu Chức Năng

## Giới thiệu
Phân Tích Dữ Liệu Chức Năng là một công cụ giúp các doanh nghiệp và tổ chức tại Việt Nam sử dụng dữ liệu để cải thiện hoạt động và tối ưu hóa quá trình ra quyết định.

## Tính năng
- Tích hợp dữ liệu từ nhiều nguồn khác nhau
- Phân tích và trực quan hóa dữ liệu theo thời gian thực
- Tạo báo cáo tùy chỉnh và chia sẻ với các bên liên quan
- Sử dụng machine learning để dự đoán và tối ưu hóa quy trình

## Công nghệ sử dụng
- Python, Pandas, Matplotlib, Scikit-learn
- Flask để xây dựng API
- MongoDB để lưu trữ dữ liệu

## Cài đặt
1. Clone repository này
2. Cài đặt các thư viện cần thiết từ `requirements.txt`
3. Chạy ứng dụng với lệnh `python app.py`

## Cách sử dụng
Sau khi chạy ứng dụng, truy cập vào `http://localhost:5000` để bắt đầu phân tích dữ liệu.
